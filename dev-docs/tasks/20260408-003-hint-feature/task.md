# Task

## タスク種別
- feature

## 背景
- 箱入り娘は手数の多いパズル（最大118手）があり、詰まったときの手がかりがない
- ソルバーの解法データを活用し、学習を支援するヒント機能を提供する

## 目的
- ゲーム画面に「ヒント」ボタンを追加し、現在の盤面から最適な次の一手を視覚的に表示する

## やること
- [ ] ゲームヘッダーに「💡 ヒント」ボタンを追加
- [ ] ボタン押下で現在の盤面を Web Worker 経由でソルバーに渡す
- [ ] 最適な次の一手のブロックと方向をハイライト表示（色変更 or 矢印）
- [ ] 計算中はローディング表示
- [ ] ヒント使用回数をカウント（クリア画面に表示検討）

## やらないこと
- 複数手先の表示
- 解法リプレイ（別タスク）
- ヒント使用によるペナルティ（星評価への影響は Phase 4 で検討）

## 完了条件
- [ ] ヒントボタン押下で次の一手がハイライトされる
- [ ] ソルバー計算中に UI がフリーズしない
- [ ] 解けない状態（ありえないが）ではエラーメッセージを表示
- [ ] typecheck, lint, unit test パス

## 制約
- Web Worker 版 Solver（Task 002）に依存
- ハイライト表示は Canvas 上で行う（DOM overlay は使わない）
