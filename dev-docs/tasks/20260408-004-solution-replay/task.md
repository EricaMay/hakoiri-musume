# Task

## タスク種別
- feature

## 背景
- パズルの最適解を見て学習したいというニーズがある
- ソルバーが最短手順を返すため、これをアニメーション再生する機能を提供する

## 目的
- ゲーム画面に「解法を見る」ボタンを追加し、最適解をステップバイステップでアニメーション再生する

## やること
- [ ] ゲーム画面に「▶ 解法」ボタンを追加
- [ ] 初期盤面からの最適解を Web Worker 経由で取得
- [ ] 解法をステップごとにアニメーション再生（既存の移動アニメーション活用）
- [ ] 再生中の一時停止 / 再開 / 停止操作
- [ ] 再生速度の調整（速い / 普通 / 遅い）

## やらないこと
- 途中盤面からの解法再生（初期盤面リセット → 再生のみ）
- リプレイデータの保存・共有

## 完了条件
- [ ] 解法ボタン押下で盤面がリセットされ、最適解がアニメーション再生される
- [ ] 再生中に一時停止・停止ができる
- [ ] ソルバー計算中に UI がフリーズしない
- [ ] typecheck, lint, unit test パス

## 制約
- Web Worker 版 Solver（Task 002）に依存
- 既存の Renderer.startAnimation を活用する
- 再生中はユーザーの操作入力を無効化する
